Tâche 1 : Analyse

L’outil UniShop est une plateforme en ligne qui sera informatisé en traitement batch en utilisant le langage Java. Les données seront stockées dans une base de données
sécurisée hébergée sur les serveurs d’UniShop. Le matériel requis est un serveur robuste pour héberger la base de données et le site web avec une connexion internet
fiable. Le système Unishop va interagir avec les systèmes de livraisons (transmission des informations de livraison, suivi colis, etc.) et de paiement en ligne
(validation méthode paiement, prise de paiement, etc.). Les utilisateurs vont interagir avec la plateforme avec des lignes de commandes.
